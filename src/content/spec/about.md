# H2RO Archive

本网站是我的公开档案站，用于整理学习、项目和生活中可以公开阅读与长期保存的材料。

这里主要收录：

- 课内学习：课程作业、实验材料、课程设计和相关文档。
- 课外学习：自学笔记、技术记录、工具使用和复盘。
- 项目记录：项目说明、实现过程、阶段成果和公开代码链接。
- 生活与兴趣：随笔、读书、游戏和其他适合公开保存的记录。

文档均会按分类归档。普通文章以 Markdown 阅读为主，PDF 文档会转换成逐页纸面预览，尽量保留原始材料的版式和可读性。部分材料来自课程、项目或阶段性学习过程，它们不一定代表标准答案，只是作为当时的记录和证据。

## 关于本站

- H2RO Archive 基于 Astro 静态站点架构构建，并在 Mizuki 博客模板的基础上改造而来，部署于 Vercel。
- Mizuki 提供了文章、归档、项目页、时间线、RSS、搜索和多种内容组件的基础骨架；在这个基础上，我把它改造成更个性化的美术风格并适当的保留和改造了部分功能。
- 视觉上刻意使用暖纸底、低饱和油墨色、细边框、轻网格、阅读型字体和克制的留白，还有部分设计灵感如壁纸来自樱之诗系列。
- 大部分笔记来自于这两年在 Obsidian 上写的私人笔记，希望未来少当懒狗继续多写一点吧。
